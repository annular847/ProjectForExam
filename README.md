WorkoutFlow — мой проект для экзамена

Что он вообще из себя представляет

WorkoutFlow — это сайт, который я сделал для популяризации ЗОЖ-а. Идея простая: помочь людям подобрать тренировки под свой возраст и телосложение. Я хотел, чтобы это выглядело современно, удобно открывалось с телефона и не пугало сложными терминами.

Сайт даёт три направления тренировок — для тех, кто хочет сжечь жир, накачать мышцы или просто поддержать форму. Плюс там есть калькулятор, который считает ИМТ и подсказывает, какая программа подойдёт лучше.

Какими функциями владеет сайт
1. Калькулятор ИМТ — вводишь свои параметры по типу (рост, вес, возраст) и он говорит, в какой ты форме. Если что-то не так — советует, на какую программу обратить внимание.
2. Три программы тренировок — для каждой есть список упражнений и рекомендация по частоте занятий.
3. Адаптивная вёрстка — я делал так, чтобы сайт нормально выглядел и на ноутбуке, и на телефоне. 
4. Смена темы — есть кнопка, которая переключает светлую и тёмную тему. Мелочь, но приятно.
5. Переключение языка — можно перевести сайт на английский. Сделал, чтобы показать, что можно работать не только с российской аудиторией.

Какие технологии использовал

Здесь я использовал языки такие как: HTML, CSS, JavaScript. Никаких фреймворков, потому что хотел показать, что понимаю основы, а не просто умею настраивать готовые библиотеки.

1. HTML5 — семантическая разметка, чтобы сайт был понятным для поисковиков и читался людьми
2. СSS3 — использовал Flexbox и Grid для адаптивности, кастомные переменные для тем
3. JavaScript — калькулятор, переключение языка, смена темы, плавный скролл
4. Font Awesome — иконки, чтобы было наглядно
5. Google Fonts — шрифт Inter, он хорошо читается

Как запустить проект

1. Скачайте файл `index.html`
2. Откройте его в браузере (просто дважды кликните по нему)
3. Всё! Никаких серверов не нужно.

Если хотите посмотреть на телефоне — можно открыть этот же файл через браузер на смартфоне или залить на GitHub Pages.

Почему именно так

Я хотел, чтобы сайт выглядел не как шаблон, а как живой продукт который будет доробатываться. В цветах использовал зелёные и жёлтые оттенки — они ассоциируются со здоровьем и энергией. Название WorkoutFlow придумал, потому что движение должно быть естественным, как поток.

На экзамене я буду показывать, как работает сайт: переключать темы, считать ИМТ, показывать адаптивность. Если что-то пойдёт не так — буду разбираться на месте, но я проверял, всё работает.
